Ejemplos de Consola JavaScript

Depuración básica con console.log()

Uso básico de console.log() para mostrar valores en la consola.

let a = 5;
let b = 6;
let c = a + b;
console.log("El valor de c es:", c); // Salida: El valor de c es: 11

Mostrar datos en formato de tabla con console.table()

console.table() es útil para visualizar arrays u objetos en formato tabular.

let personas = [
  { Nombre: "Juan", Apellido: "González" },
  { Nombre: "María", Apellido: "López" },
  { Nombre: "Pedro", Apellido: "Martínez" }
];
console.table(personas);

Manejo de errores con try-catch

Ejemplo de cómo capturar y manejar errores en JavaScript.

try {
  let numero = 0;
  console.log("Intentando dividir por cero...");
  let resultado = 55 / numero; // Esto genera una excepción
  console.log("Resultado:", resultado);
} catch (error) {
  console.error("Ocurrió un error:", error.message);
}

Mensajes informativos con console.info()

console.info() se usa para mensajes informativos.

console.info("Esto es un mensaje informativo.");

Advertencias con console.warn()

console.warn() muestra mensajes de advertencia.

console.warn("¡Advertencia! Algo podría no estar bien.");

Errores con console.error()

console.error() muestra mensajes de error.

console.error("Esto es un mensaje de error.");

Abra la consola del navegador (F12) para ver los resultados de los ejemplos.
